Entry Lesson 08

What time is it?

何時ですか？

Today's GOAL

To be able to ask what time it is, and understand the basic imperative forms.
時間を聞くことができる / 命令形の基本形を理解することができる。

Today's Phrase

Wake up!
おきなさい。
What time is it?
何時ですか？
It’s 3 o’clock.
3時です。

Go over the example with students.
Example （例）:
Run. （走れ）>> Don’t run. （走らないで。）>>　Please don’t run. （走らないで下さい。）
Read the original verb (for example *cry* for the first one)and have students repeat after you.

Verb(動詞) Cry
Teacher Don't cry.
Teacher Please don't cry.
(泣かないで下さい。)
Verb(動詞) Attack.
Teacher Don't attack.
(攻撃するな。)

(攻撃しないで下さい。)
