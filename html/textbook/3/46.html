Middle Lesson 46

If the fireman had been late,she'd not have been saved.

Today's GOAL

To be able to talk about what might have happened in the past.
過去に起こったかもしれないことについて話せるようになる。

Today's Phrase

If the fireman had been late,the girl wouldn't have been saved.
If he hadn't leaped from the window,the flames would have reached him.

Third Conditional ( If S had Vpp, S would have Vpp)

The if-clause states an action that did not take place in the past. However, if the action in the if-clause had actually taken place, it is quite probable that the action in the result clause would have taken place too. We use the third conditional to talk about what might have happened.

If the fireman had been late, the girl would not have been saved.
(if-clause) (the result clause)

Short Conversation

Directions: Read the following conversation with your teacher.

Sarah and Lydia are talking about the fire which occurred at the inn in their neighborhood.
Lydia Hey, did you hear about the fire last night at Sunday Inn ?
Sarah Yeah, the owner was napping by the fireplace, and somehow a fire started, right?
Lydia Right! If he hadn't leaped from the window, the flames would have reached him.
Sarah That's so terrifying. Was there anyone else in there?
Lydia There were a few customers there. Everyone escaped except for the little girl who was left alone. A fireman dove back into the burning building in a heroic attempt to save the child.
Sarah Oh wow! Was the girl saved?
Lydia Yeah I heard so. The fire engines were speeding by with their sirens wailing all night so I couldn't sleep much.
Sarah Neither could I. But I'm glad that the little girl was saved.
Lydia If the fireman had gone in just a minute later, she wouldn't have been saved. Thank God both of them are safe.

Exercise

Directions: Complete the sentences using
3rd conditional tense (if I had (not) Vpp, I would (not) have Vpp.)

Example: If you had been there, we would have enjoyed the party much more. (be / enjoy)
If you had told me it was dangerous, I wouldn’ t have gone there all by myself. (tell / go)
  1. If the movie _____ a happy ending, they _____ it. ( have / love )
    had     would have loved
  2. If they _____ invited, they _____ to the meeting. ( be / go )
    had been     would have gone
  3. If we _____ the bus, we _____ to work on time. ( miss / get )
    had missed    wouldn't have got/gotten
  4. If he _____ what he'd done, he _____ to her. ( regret / apologize )
    had regretted    would have apologized
  5. If she _____ the beach was dangerous, she _____ there. ( know / swim )
    had known    wouldn't have swum
  6. I _____ married him if he _____ to me last year. ( get / propose )
    would have got/gotten    had proposed
  7. He _____ his girlfriend a ride if his car _____ broken. ( give / be )
    would have given    hadn't been
  8. We _____ more if there _____ more gas in the car. ( drive / be )
    would have driven    had been
  9. They _____ with each other if he _____ about the incident. ( fight / talk )
    wouldn't have fought    hadn't talked
