Practice

Directions : Repeat the 3 answers to the key phrase.

Question
This design looks good.

Answer 1
Yes, I agree.

イェス, アイ アグリー.
(ええ、同感です。)

Answer 2

アイ ドンットゥ リアリー ライキットゥ.
(あまり私の好みではありません。)

ザッツ サムスィング ニュー.
(新しいですね。)

More tips

"look"には「見る、～に見える」と意味があります。
そのため、"look(s) good"は、「素敵に見える、良く見える」という意 味になります

Page 7

Practice

Directions : Repeat the 3 answers to the key phrase.

先ほど練習したキーフレーズに対する3つの答えをリピートしてみましょう！

Question
Do you have a smaller size?

Answer 1
Just a moment.

ジャスタ モーメントゥ.


(少々お待ちください。)

Answer 2
It’s the smallest size we have.

イッツ ザ スモーレストゥ サイズ ウィー ハヴ.
(そちらが一番小さなサイズです。)

Answer 3
I’m sorry, we don’t.

アイム ソーリー, ウィー ドンットゥ.
(申し訳ありませんが、ございません。)
Page 8

More tips

“Do you have ～?“は、相手が特定のものを持っているかどうかを 尋ねる表現です。 ものだけでなく、考え(アイディアや気持ちなど)や 状態(熱や風邪など)などに対しても、幅広く使うことのできるフレ ーズです。

    そちらが一番小さなサイズです。

  • C. I’m sorry, we don’t.

    申し訳ありませんが、ございません。

Directions : Repeat the 3 answers to the key phrase.

(よく分からないな。)

Page 11

More tips
